## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/erfanbanaei/erfanbanaei_ir.git
    ```
2. Navigate to the project directory:
    ```bash
    cd erfanbanaei_ir
    ```
3. Run the project:
    ```bash
    python3 -m http.server
    ```
